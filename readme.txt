== Description ==

What Am I Listening To?

A Wordpress plugin to show what your are listening from Last.FM API

This Plugin connecting to Last.FM Api with you Api Key defined in settings.

The Last.fm API allows anyone to build their own programs using Last.fm data, whether they're on the web, the desktop or mobile devices.

## Shortcode:

[wailt number='1']

### Shortcode Attributes:

**number**: number of tracks to show

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/what-am-i-listening-to` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go to Settings page and enter your Last.FM Api Key
4. Use the shortcode or widget described in this README

== Frequently Asked Questions ==

= How is it working? =

This module is working with connecting to Last.fm with your Api key.
